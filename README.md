## Installation
- git clone or download the `everblush.ini file` from theme dir. 
- add it to your polybar configuration directly.
- Include the `everblush.ini` in your `config.ini`. 
```sh 
[global/wm] 
include-file = $HOME/.config/polybar/everblush.ini
``
